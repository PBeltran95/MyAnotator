# MyAnotator app
A noteApp


## Features used:
- Room
- Coroutines
- MVVM
- ViewModel & LiveData
- ViewBinding
- Dependency Injection with Dagger Hilt
- Extension functions
- RecyclerView
- Fragments
- Navigation Component
- ConstraintLayout, MaterialCardView
-DiffUtils
-RecyclerViewAnimations
-SwipeAnimations
