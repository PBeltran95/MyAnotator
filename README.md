# MatchDogsApp
An app in which you can adopt a dog. Take as much dogs as you can in your phone!


## Features used:
- Room
- Coroutines
- MVVM
- ViewModel & LiveData
- ViewBinding
- Dependency Injection with Dagger Hilt
- Extension functions
- RecyclerView
- Fragments
- Navigation Component
- ConstraintLayout, MaterialCardView
-DiffUtils
-RecyclerViewAnimations
-SwipeAnimations
